# [PRO-801] FE: 인터랙티브 고도표(Recharts) 및 세그먼트 편집 로직 구현

## 🎯 목표
코스의 고도 변화를 그래프로 보여주고, 사용자가 그래프 상에서 구간을 선택/분할/병합할 수 있는 인터페이스를 제공한다.

## ✅ To-Do List
- [ ] `recharts` 라이브러리 설치
- [ ] `ElevationChart` 컴포넌트 구현 (AreaChart 활용)
- [ ] 3D 지도와 고도표 간 마우스 호버 싱크(Sync) 구현
- [ ] 구간 편집 인터랙션:
    - [ ] **Click-to-Split:** 차트 클릭 시 해당 지점에서 세그먼트 분할
    - [ ] **Drag-to-Merge:** 브러시 도구로 드래그하여 구간 병합
- [ ] 편집된 세그먼트 정보를 Zustand 스토어(`segments`)에 반영

## 🏁 Definition of Done (DoD)
- 고도표에서 구간을 나누거나 합치면, 3D 지도상의 색상도 즉시 변경되어야 한다.
- 편집된 세그먼트 정보가 GPX 메타데이터로 저장(Export) 가능해야 한다.
