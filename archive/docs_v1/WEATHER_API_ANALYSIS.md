# [보고서] 사이클링 시뮬레이션 고도화를 위한 기상 API 도입 검토 (최종)

## 1. 후보 선정 배경: 왜 이 두 서비스인가?
기존 검토 대상 4개 업체 중 Windy와 Visual Crossing은 아래와 같은 비즈니스 및 기술적 제약으로 인해 우선순위에서 제외되었습니다.

*   **Windy 제외 사유:** 무료 테스트 시 실제와 다른 변형된 데이터를 제공하여 물리 엔진 검증이 불가능하며, 과거 기상 데이터를 아예 제공하지 않음.
*   **Visual Crossing 제외 사유:** 역사적 데이터는 풍부하나, 현재 개발 중인 시뮬레이션 엔진과의 API 통합 편의성 및 실시간 데이터 업데이트 속도 면에서 OWM과 Open-Meteo가 더 적합함.

따라서, 과거 데이터(Historical Data)의 접근성과 고도별 풍속 데이터를 갖춘 **Open-Meteo**와 **OpenWeatherMap**을 최종 후보로 비교 분석하였습니다.

---

## 2. 최종 후보 서비스 비교 상세

### A. Open-Meteo (연구 및 프로토타이핑 최적화)
*   **강점:** API 키 없이(No Key) 무료 티어에서 즉시 과거 데이터(Historical Weather API)를 테스트할 수 있어, 초기 리서치 비용이 전혀 들지 않습니다.
*   **가격:** 월정액제로 비용이 투명하고 고정적입니다 (Professional 기준 월 500만 건 호출 가능).
*   **제공 범위:** 1km 고해상도 로컬 모델을 포함하며, 10m/80m/120m/180m 등 주요 고도별 풍속 데이터를 제공합니다.
*   **라이선스 유의사항:** 상용 도입 시에도 반드시 출처(Attribution)를 표기해야 하는 CC BY 4.0 라이선스를 따릅니다. 또한, 유료 단계인 Standard 플랜에서는 과거 데이터가 제외되어 있어, 상용 서비스 시에는 반드시 Professional 플랜을 도입해야 합니다.

### B. OpenWeatherMap (상용화 및 비즈니스 확장 최적화)
*   **강점:** One Call API 3.0을 통해 47년 이상의 방대한 과거 데이터를 제공하며, 매일 1,000건까지는 무료로 호출할 수 있어 소규모 검증에 유리합니다.
*   **가격:** 사용한 만큼 지불하는 종량제(Pay-as-you-call) 또는 기업형(Enterprise) 구독 모델을 선택할 수 있습니다.
*   **제공 범위:** 10m에서 500m 사이의 **모든 고도(any hub heights)**에 대한 풍향/풍속 데이터를 제공하여 시뮬레이션의 정밀도를 극대화할 수 있습니다.
*   **라이선스 관리:** Enterprise 등급 도입 시 출처 표기(Attribution) 의무가 완전히 사라지므로, 자사 브랜드의 독립성을 유지하는 데 가장 유리합니다.

## 3. 핵심 비교 요약표

| 비교 항목 | Open-Meteo | OpenWeatherMap |
| :--- | :--- | :--- |
| **초기 테스트** | **API 키 없이 과거 데이터 사용 가능** | API 키 발급 후 일 1,000건 무료 |
| **과거 데이터** | 무료 및 Professional에서만 제공 | One Call 3.0에서 47년치 제공 |
| **풍속 데이터** | 특정 고도별(10~180m) 제공 | **10~500m 사이 모든 고도 제공** |
| **비용 관리** | 고정 월정액 (Standard/Pro) | 종량제 또는 고가 기업용 플랜 |
| **라이선스** | **출처 표기 필수 (CC BY 4.0)** | 기업용 도입 시 출처 표기 면제 |

---

## 4. 최종 검토 의견 및 향후 계획

1.  **단기적 관점 (테스트/리서치):**
    *   **Open-Meteo**의 무료 티어를 활용하여 별도의 가입 절차나 비용 지불 없이 엔진의 기본 로직과 과거 기상 데이터의 정합성을 즉시 검증하겠습니다.

2.  **중장기적 관점 (서비스 상용화):**
    *   사용자 수가 급증하고 서비스 내에서 브랜드 노출을 최소화해야 하는 시점에는 **OpenWeatherMap**의 비즈니스 라이선스로 전환하는 것이 유리합니다. 특히 고도별 풍속의 정밀도(10~500m 자유 선택) 면에서 OWM이 시뮬레이션 품질 우위를 점할 수 있습니다.

**결론:** 현재는 **Open-Meteo**의 개방된 과거 데이터를 통해 리서치 속도를 높이고, 최종 서비스 품질과 브랜드 정책에 따라 **OpenWeatherMap으로의 확장을 고려하는 전략**을 추천드립니다.
